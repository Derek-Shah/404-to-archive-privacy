# 404 to Archive Privacy Policy

Effective date: 2026-02-14

404 to Archive does not collect, sell, or transmit personal data for analytics, advertising, profiling, or telemetry.

## Data we store
The extension stores only local settings in `chrome.storage.local`:
- automatic redirect setting
- soft-404 detection setting
- domain blacklist entries

## Data we do not store
- browsing history
- page content
- form input
- account identifiers
- IP address logs

## Network behavior
The extension contacts `web.archive.org` only when:
- the user chooses “Open archive”, or
- the user has explicitly enabled automatic redirects.

## Sharing
We do not sell or share user data with third parties.
